Project Description



Pathologists have to validate their tests on a regular basis. The weekly schedule of validation combined with the number of different tests to validate generates a lot of paperwork. Currently this paperwork is manually filled out, handed off to the doctor, filled out by the doctor, and then returned. This takes up time and effort that could be better spent actually running tests or doing other important things.

Our goal is to create a web app that can replace the paper forms that are currently in use. This will help streamline the bureaucratic process associated with running calibration tests. Currently this project is being developed for the molecular histology lab here at UNC where it will be used by pathologists and their technicians who are validating tests.

In the ongoing wake of covid-19 it is clearer than ever how much work there is for doctors and healthcare workers. Any chance to streamline their work or cut down on the time they spend on bureaucratic tasks has a real world impact. Hopefully this project will make it easier for the UNC molecular histology lab to validate their tests and thus let them spend more time on actually running tests.
